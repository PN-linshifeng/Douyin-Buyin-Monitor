# 达人卷嗅探

扩展脚本：帮我在url：/dashboard/merch-picking-library/merch-promoting， 在右下角加个“达人卷嗅探”， 监听浏览器url不是/dashboard/merch-picking-library/merch-promoting就删除按钮，点击按钮操作如下：

- 第一步: 加橱窗，product_id的值是传参数
fetch("https://buyin.jinritemai.com/pc/selection/window/pmt/add", {

  "body": "product_id=3632218040395005570&item_type=4&pick_first_source=%E7%99%BE%E5%BA%94&pick_second_source=%E9%80%89%E5%93%81%E5%B9%BF%E5%9C%BA&pick_third_source=category_recommend&pick_source_id=",
  "method": "POST",

});

- 第二步：请求达人卷列表
  promotion_name_or_id是传参数，_是时间戳。根据参数promotion_name_or_id 在返回的数据找
  promotion_id==promotion_name_or_id或者promotion_name_or_id==promotion_id，如果该条的reject_reason==='该商家不允许达人配置达人券', 达人卷嗅探 按钮的文字改成“该商家不允许达人配置达人券”，否则显示：“参加，价格：discount_price_display，佣金率：cos_ratio_display”
fetch("https://buyin.jinritemai.com/api/buyin/marketing/anchor_coupon/promotion_list?_bid=mcenter_buyin&_luckybag_bid=buyin_luckybag&_=1766420092741&s=485333&promotion_name_or_id=3723171937199981382&page=1&size=10&search_type=0&need_channel=true", {

  "method": "GET",
  "mode": "cors",
  "credentials": "include"
});
